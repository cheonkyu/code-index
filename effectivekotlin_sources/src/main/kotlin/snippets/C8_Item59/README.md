# Item 59: `let`, `run`, `with`, `apply`, `also` 스코프 함수를 적절하게 사용하세요

안녕하세요! 코틀린 프로그래밍을 더욱 효과적으로 만들어 줄 Item 59에 대해 함께 알아볼 거예요. 코틀린의 스코프 함수들은 코드를 간결하고 읽기 쉽게 만들어주는 강력한 도구랍니다. 각각의 함수들이 어떤 특징을 가지고 있으며, 어떤 상황에서 사용하는 것이 가장 좋을지 자세히 살펴볼까요?

## 스코프 함수란 무엇인가요?

스코프 함수들은 객체를 `람다` 표현식의 인자로 전달하여, 객체의 멤버에 안전하게 접근하고 코드를 블록으로 감싸는 역할을 합니다. 이 함수들을 사용하면 묵시적으로 객체를 인자로 전달하기 때문에 코드가 훨씬 깔끔해지고 가독성이 높아져요.

## 주요 스코프 함수 종류와 특징

코틀린에는 `let`, `run`, `with`, `apply`, `also`라는 다섯 가지 주요 스코프 함수가 있습니다. 각각의 함수는 객체를 인자로 받아 블록 내에서 사용할 수 있도록 하고, 반환 값의 방식이 조금씩 다르답니다.

1. **`let`**:
   - 객체를 `it`이라는 이름으로 블록에 전달합니다.
   - 블록의 마지막 표현식의 결과값을 반환합니다.
   - `null` 안전하게 객체를 다루는 데 유용합니다. 객체가 `null`이 아니라면 블록을 실행하고, `null`이라면 블록을 건너뜁니다.

   ```kotlin
   val message: String? = "Hello, Kotlin!"

   message?.let {
       println("Message length: ${it.length}")
   }
   ```

2. **`run`**:
   - 객체를 `this`로 블록에 전달합니다.
   - 블록의 마지막 표현식의 결과값을 반환합니다.
   - 주로 객체 생성 후 초기화 작업을 수행하거나, 객체의 기능을 수행하는 데 사용됩니다.

   ```kotlin
   data class Person(var name: String, var age: Int)

   val person = Person("Alice", 30).run {
       age += 1
       println("Name: $name, Age: $age")
       this // Person 객체 반환
   }
   ```

3. **`with`**:
   - 객체를 `this`로 블록에 전달합니다.
   - 블록의 마지막 표현식의 결과값을 반환합니다.
   - 여러 개의 속성을 설정하거나, 객체의 여러 기능을 순차적으로 호출할 때 유용합니다.

   ```kotlin
   val builder = StringBuilder()
   with(builder) {
       append("This is a ")
       append("string builder.")
       println(toString())
   }
   ```

4. **`apply`**:
   - 객체를 `this`로 블록에 전달합니다.
   - 객체 자체를 반환합니다.
   - 주로 객체를 생성하고 초기화하는 데 사용됩니다. 객체의 속성을 설정하고 객체 자체를 반환하기 때문에, 체이닝(chaining)을 통해 코드를 간결하게 만들 수 있습니다.

   ```kotlin
   val person = Person("Bob", 25).apply {
       name = "Charlie"
       age = 35
   }
   println(person) // Person(name=Charlie, age=35)
   ```

5. **`also`**:
   - 객체를 `it`으로 블록에 전달합니다.
   - 객체 자체를 반환합니다.
   - 객체의 상태를 변경하지 않고 추가적인 작업을 수행할 때 유용합니다. 예를 들어, 객체를 로깅하거나, 부수적인 효과를 발생시키는 데 사용할 수 있습니다.

   ```kotlin
   val person = Person("David", 40).also {
       println("Person created: $it")
   }
   println(person) // Person(name=David, age=40)
   ```

## 어떤 함수를 사용해야 할까요?

각 스코프 함수는 고유한 특징을 가지고 있기 때문에, 어떤 함수를 사용해야 할지 상황에 따라 판단해야 합니다. 다음 가이드라인을 참고해 보세요.

* **`let`**: `null` 안전하게 객체를 처리해야 할 때
* **`run`**: 객체의 기능을 실행하거나, 객체 생성 후 초기화 작업을 수행할 때
* **`with`**: 객체의 여러 기능을 순차적으로 호출할 때
* **`apply`**: 객체의 속성을 설정하고 객체 자체를 반환해야 할 때 (객체 구성)
* **`also`**: 객체의 상태를 변경하지 않고 부수적인 작업을 수행할 때 (로깅, 디버깅)

## 핵심 요약

* `let`, `run`, `with`, `apply`, `also`는 각각 고유한 특징과 반환 값을 가지고 있습니다.
