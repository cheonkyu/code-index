# 24. 리스트(Lists) 이해하기: 초보자 가이드

안녕하세요! 코틀린 프로그래밍을 처음 접하시는 분들을 위해 리스트(Lists)에 대해 쉽게 설명해드릴게요. 리스트는 정말 쉽게 생각하면 여러 가지 객체들을 한데 모아두는 용기라고 볼 수 있어요. 이런 용기는 코틀린에서는 `Collections` 또는 더 간단히 `리스트`라고 부르죠. 특별히 이번 챕터에서는 리스트를 주로 다루어 보면서 기본적인 사용법부터 다양한 기능까지 배워볼게요.

## 리스트 생성하기

리스트를 만드는 방법은 정말 간단해요! `listOf()`라는 함수를 사용하면 되는데요, 이 함수에 원하는 객체들을 넣어주면 돼요. 예를 들어, 정수 리스트를 만드는 코드는 다음과 같아요:

```kotlin
// 예제 코드: 정수 리스트 생성
fun main() {
    // 정수 리스트 생성
    val ints = listOf(99, 3, 5, 7, 11, 13)
    
    // 리스트 출력 (리스트는 [ ]로 표시)
    println("리스트 내용: ${ints}") // 출력: [99, 3, 5, 7, 11, 13]
    
    // 리스트의 요소를 하나씩 접근하기
    var result = ""
    for (i in ints) {
        result += "$i "  // 각 요소를 문자열로 연결
    }
    println("각 요소 연결: $result") // 출력: 99 3 5 7 11 13
    
    // 인덱스를 이용해 요소 접근하기
    println("인덱스 4의 요소: ${ints[4]}") // 출력: 11
}
```

### 주요 포인트 요약
1. **리스트 생성**: `listOf()` 함수로 쉽게 생성 가능합니다. 예를 들어 `listOf(99, 3, 5)`처럼 객체들을 넣어주세요.
2. **리스트 출력**: 리스트는 `[ ]` 괄호로 표시됩니다. 예: `println(ints)`는 `[99, 3, 5, ...]`를 출력합니다.
3. **요소 접근**: `for` 루프를 사용하면 리스트의 각 요소를 쉽게 순회할 수 있습니다. 인덱스를 사용해 특정 위치의 요소를 가져올 수도 있어요.
   - **주의**: 인덱스는 0부터 시작합니다. 즉, 첫 번째 요소는 인덱스 0에 위치해 있어요! 그래서 인덱스 4는 리스트의 다섯 번째 요소를 가리킵니다.

## 리스트의 다양한 기능

리스트는 다양한 기능을 제공해요. 몇 가지 주요 기능들을 살펴보죠:

### 합계와 정렬
```kotlin
// 예제 코드: 리스트의 합계와 정렬
fun main() {
    val doubles = listOf(1.1, 2.2, 3.3, 4.4)
    println("리스트 합계: ${doubles.sum()}") // 출력: 11.0
    
    val strings = listOf("Twas", "Brillig", "And", "Slithy", "Toves")
    println("정렬된 리스트: ${strings.sorted()}") // 출력: ["And", "Brillig", "Slithy", "Toves", "Twas"]
    println("역순 리스트: ${strings.reversed()}") // 출력: ["Toves", "Slithy", "And", "Brillig", "Twas"]
    println("첫 번째 요소: ${strings.first()}") // 출력: Twas
    println("마지막 두 요소: ${strings.takeLast(2)}") // 출력: ["Slithy", "Toves"]
}
```

### 주요 포인트 요약
1. **합계**: `sum()` 메서드로 리스트의 모든 요소를 합산할 수 있어요.
2. **정렬**: `sorted()` 메서드는 새로운 정렬된 리스트를 반환합니다. 원본 리스트는 변경되지 않아요.
3. **역순**: `reversed()`로 리스트를 역순으로 만들 수 있어요. 역시 원본 리스트는 그대로 유지됩니다.

### 타입 지정하기
리스트가 어떤 타입의 객체를 포함할지 명시적으로 지정할 수도 있어요:
```kotlin
// 예제 코드: 타입 지정
fun main() {
    // 타입 추론
    val numbers = listOf(1, 2, 3)
    val strings = listOf("one", "two", "three")
    
    // 명시적 타입 지정
    val numbers2: List<Int> = listOf(1, 2, 3)
    val strings2: List<String> = listOf("one", "two", "three")
    
    println("타입 추론 리스트: $numbers, 명시적 타입 지정 리스트: $numbers2") // 출력: [1, 2, 3], [1, 2, 3]
    println("타입 추론 문자열 리스트: $strings, 명시적 타입 지정 문자열 리스트: $strings2") // 출력: ["one", "two", "three"], ["one", "two", "three"]
}
```

### 주요 포인트 요약
1. **타입 추론**: 코틀린은 자주 타입을 자동으로 추론해요. 하지만 때로는 명시적으로 타입을 지정해주는 것이 더 명확할 수 있어요.
2. **타입 파라미터**: `List<타입>` 형태로 타입을 지정하면 더 구체적인 타입 정보를 제공할 수 있어요. 예를 들어 `List<Int>`는 정수 리스트를 의미합니다.
