## 클래스란 무엇인가요?

클래스는 당신이 해결하려는 문제의 일부를 표현하는 개념적인 '타입'이라고 생각하면 됩니다. 예를 들어, 동물원 프로그램을 만드는 상황을 생각해보세요. 동물들을 종류별로 분류하고 싶다면, 각 동물의 특성과 행동을 클래스로 정의할 수 있어요.

### 기본 클래스 정의하기

먼저 간단한 예제를 통해 클래스를 만드는 방법을 보여드릴게요:

```kotlin
// CreatingClasses/Animals.kt
class Giraffe // 기린 클래스 정의
class Bear    // 곰 클래스 정의
class Hippo   // 하마 클래스 정의

fun main() {
    // 각 클래스의 객체 생성
    val g1 = Giraffe()       // 기린 객체 1
    val g2 = Giraffe()       // 기린 객체 2
    val b = Bear()           // 곰 객체
    val h = Hippo()          // 하마 객체

    // 객체 출력 (주소 출력 포함)
    println(g1)             // Giraffe@28d93b30 (예시 주소)
    println(g2)             // Giraffe@1b6d3586 (예시 주소)
    println(h)              // Hippo@4554617c (예시 주소)
    println(b)              // Bear@74a14482 (예시 주소)
}
```

### 클래스 이름 규칙
- 클래스 이름은 대문자로 시작해야 합니다 (예: `Giraffe`, `Bear`, `Hippo`).
- 일반적으로 클래스 이름은 명사로 짓는 것이 좋습니다.
- 변수나 상수 이름은 소문자로 시작하는 것이 관례입니다 (예: `g1`, `b`).

### 객체와 클래스의 관계
- **클래스**는 설계도나 템플릿 같은 역할을 합니다. 예를 들어, `Giraffe` 클래스는 모든 기린에 대한 공통 특성을 정의합니다.
- **객체**는 실제 존재하는 것으로, 클래스의 인스턴스입니다. `g1`, `g2`, `b`, `h`는 각각 독립적인 기린과 곰, 하마를 나타냅니다.

## 멤버 함수 이해하기

클래스 내에서 정의된 함수는 **멤버 함수**라고 부르며, 이는 클래스와 밀접하게 연관되어 있습니다. 함수 호출 시 객체 이름 뒤에 `.`을 붙여 함수를 호출합니다.

### 예제: 멤버 함수 호출

```kotlin
// CreatingClasses/Dog.kt
class Dog {
    fun bark() = "yip!"  // 멤버 함수 bark() 정의
}

fun main() {
    val dog = Dog()      // Dog 객체 생성
    // bark() 함수 호출 (멤버 함수)
    // dog.bark() 처럼 호출하면 됩니다.
}
```

```kotlin
// CreatingClasses/Cat.kt
class Cat {
    fun meow() = "mrrrow!"  // 멤버 함수 meow() 정의
}

fun main() {
    val cat = Cat()      // Cat 객체 생성
    val m1 = cat.meow()  // 멤버 함수 호출 및 결과 저장
    println(m1)         // 출력: mrrrow!
}
```

### 주요 포인트 요약
1. **클래스 정의**: `class 클래스명 { ... }`로 클래스를 정의합니다.
2. **객체 생성**: `클래스명()` 형태로 객체를 생성합니다.
3. **멤버 함수 호출**: 객체 이름 뒤에 `.`을 붙여 함수를 호출합니다 (`객체명.함수명()`).
4. **기능 구분**: 클래스 내의 함수는 클래스에 소속된 기능을 나타내며, 객체별로 동작합니다.
