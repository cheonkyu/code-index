# 08. 문자열 템플릿 (String Templates)

안녕하세요, 초보자 여러분! 이번 챕터에서는 Kotlin에서 문자열을 만들고 조작하는 쉬운 방법 중 하나인 **문자열 템플릿**에 대해 배워볼게요. 문자열 템플릿은 코드로 문자열을 생성하는 편리한 방법이랍니다. 이해하기 쉽게 단계별로 설명해 드릴게요.

## 기본 개념: `$` 기호 사용하기

### 문자열 내 변수 삽입하기
`$` 기호를 사용하면 문자열 내에 변수 값을 쉽게 삽입할 수 있어요. `$` 뒤에 변수 이름을 쓰면 그 변수의 값이 문자열에 대입됩니다.

**예제 코드:**

```kotlin
fun main() {
    val answer = 42
    println("Found $answer!")  // 출력: Found 42!
    println("printing a $1")  // 출력: printing a $1 (변수가 없으면 그대로 출력)
}
```

**해설:**
- `"Found $answer!"`에서 `$answer` 부분은 `answer` 변수의 값인 `42`로 대체됩니다.
- `"printing a $1"`에서 `$1`은 식별자가 아니므로 그대로 출력됩니다. 변수가 없으면 Kotlin은 특별한 처리를 하지 않아요.

## 간단한 문자열 연결: `+` 연산자 사용하기

문자열을 연결할 때는 `+` 연산자를 사용할 수도 있어요. 하지만 이 방법은 코드가 길어질 수 있으니, 간단한 경우에만 사용해보세요.

**예제 코드:**

```kotlin
fun main() {
    val s = "hi\n"  // 줄바꿈 문자 `\n`
    val n = 11
    val d = 3.14
    println("first: " + s + "second: " + n + ", third: " + d)
    // 출력:
    // first: hi
    // second: 11, third: 3.14
}
```

**해설:**
- 여러 문자열을 연결할 때 `+` 연산자를 사용하면 됩니다. 하지만 코드가 복잡해지면 가독성을 해칠 수 있으니 주의하세요!

## 표현식을 포함한 문자열 템플릿

문자열 내에 표현식을 포함시키면 해당 표현식의 결과 값이 문자열로 변환되어 삽입됩니다.

**예제 코드:**

```kotlin
fun main() {
    val condition = true
    println(
        "${if (condition) 'a' else 'b'}"  // 출력: a
    )
    val x = 11
    println("$x + 4 = ${x + 4}")  // 출력: 11 + 4 = 15
}
```

**해설:**
- `${if (condition) 'a' else 'b'}` 부분은 `condition`이 참이면 `'a'`, 거짓이면 `'b'`를 반환합니다.
- `${x + 4}`는 표현식의 결과 값인 `15`를 문자열로 변환하여 삽입합니다.

## 특수 문자 삽입하기

문자열 내에 특수 문자 (예: 인용 부호 `'`)가 포함되어 있을 때는 `\`로 이스케이프 해야 합니다. 또는 삼중 따옴표(`"""`)를 사용할 수도 있어요.

**예제 코드:**

```kotlin
fun main() {
    val s = "value"
    println("s = \"$s\"")  // 출력: s = "value"
    println("""s = "$s"""")  // 출력: s = "value"
}
```

**해설:**
- `"$s"`에서 작은따옴표 안에 작은따옴표를 포함시키려면 `\`로 이스케이프하거나 삼중 따옴표를 사용합니다.
- 삼중 따옴표는 문자열 내부의 인용 부호를 쉽게 다룰 수 있게 해줍니다.

## 핵심 요약

- **문자열 템플릿**: `$` 기호를 사용해 변수 값을 문자열에 삽입합니다.
- **표현식 포함**: `${표현식}`으로 표현식 결과를 문자열에 삽입합니다.
- **특수 문자 처리**: `\` 이스케이프 또는 삼중 따옴표 사용으로 특수 문자를 안전하게 삽입합니다.
- **연결 연산자**: 간단한 연결에는 `+` 연산자를 사용할 수 있지만, 가독성을 위해 주의하세요.
