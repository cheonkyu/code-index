# Hello, World! 코틀린 시작하기

안녕하세요! 코틀린 프로그래밍 세계에 오신 것을 환영합니다. 오늘은 코틀린 프로그래밍의 기본을 이해하는 데 있어 가장 중요한 프로그램 중 하나인 "Hello, World!"를 함께 만들어 보겠습니다. 초보자도 쉽게 따라 할 수 있도록 단계별로 설명드리겠습니다.

## 1. 빈 프로그램 살펴보기

### 기본 구조
코틀린 프로그램은 `main()` 함수를 중심으로 구성됩니다. `main()` 함수는 프로그램의 시작 지점을 나타내며, 이 함수가 실행되면 프로그램이 동작합니다.

#### 예시 코드:
```kotlin
// HelloWorld/EmptyProgram.kt
fun main() {
    // 프로그램 코드 여기...
}
```

- **주석(Comments)**: 코드 위에 `//`로 시작하는 텍스트는 주석입니다. 코틀린은 이 주석 부분을 무시하고 실행하지 않습니다. 예를 들어,
  ```kotlin
  // 이 코드는 실행되지 않습니다
  ```

- **함수(Function)**: `fun` 키워드는 함수를 정의합니다. 함수는 코드 블록을 이름으로 실행할 수 있게 해줍니다. `main()` 함수는 특별한 이름으로, 프로그램이 실행될 때 자동으로 호출됩니다.
  ```kotlin
  fun main() {
      // 여기에 코드 작성
  }
  ```

## 2. "Hello, World!" 출력하기

### 출력 함수 사용하기
"Hello, World!"를 화면에 출력하기 위해 `println()` 함수를 사용합니다.

#### 예시 코드:
```kotlin
// HelloWorld/HelloWorld.kt
fun main() {
    println("Hello, world!")
}
```

- **println() 함수**: `println()`은 문자열을 화면에 출력하는 함수입니다.
  - **문자열 정의**: 문자열은 쌍따옴표(`"`)로 묶어 정의합니다. 예를 들어, `"Hello, world!"`는 문자열을 나타냅니다.
  - **출력 동작**: `println()` 함수는 매개변수를 받아 화면에 출력한 후 다음 줄로 커서를 이동시킵니다. 만약 같은 줄에 유지하고 싶다면 `print()` 함수를 사용할 수 있습니다 (코틀린에서는 `print()` 대신 `println()`이 주로 사용됩니다).

### 실행 결과
위 코드를 실행하면 다음과 같은 결과가 화면에 나타납니다:
```
Hello, world!
```

## 주요 개념 요약
- **주석(Comments)**: 코드 이해를 돕기 위해 사용되며, 코틀린은 `//`로 시작하는 텍스트를 무시합니다.
- **함수(Functions)**: 코드 블록을 이름으로 실행할 수 있게 해줍니다. `fun main()`은 프로그램의 시작점입니다.
- **println()**: 문자열을 화면에 출력하는 기본 함수입니다. 문자열은 쌍따옴표로 묶습니다.
