# 11. 반복문으로 반복 작업하기: `while`과 `do-while` 루프 사용법

안녕하세요! 코틀린 프로그래밍을 처음 접하는 분들을 위해 이 챕터에서는 반복 작업을 하는 데 매우 유용한 `while` 루프와 `do-while` 루프에 대해 배워볼게요. 컴퓨터는 반복 작업에 정말 탁월하잖아요! 그럼 함께 핵심 개념을 익혀볼까요?

## `while` 루프 이해하기

`while` 루프는 특정 조건이 참(true)일 때 코드 블록을 계속 반복해 실행하는 구조예요. 쉽게 말해, 조건이 만족될 때까지 계속해서 작업을 반복하는 거죠.

### 기본 구조
```kotlin
while (참조건) {
    // 반복할 코드 블록
}
```

### 예제 코드 설명
```kotlin
// 조건 함수 정의: 숫자가 100 미만일 때 참(true)을 반환
fun condition(i: Int) = i < 100

// 메인 함수
fun main() {
    var i = 0  // 변수 i를 초기화
    while (condition(i)) {  // 조건이 참이면 반복 시작
        print(".")  // 출력: 점(.)
        i += 10  // i에 10을 더해 다음 반복으로 이동
    }
}
/* 출력 결과:
..........
*/

**핵심 개념 요약:**
- **참조건**: 루프를 계속할지 결정하는 Boolean 표현식입니다.
- **반복 코드 블록**: 참조건이 참인 동안 계속 실행되는 코드들입니다.
- `i += 10`: `i`에 10을 더하는 간단한 증가 연산자로, 코드를 간결하게 만듭니다.

## `do-while` 루프 이해하기

`do-while` 루프는 `while` 루프와 비슷하지만, 최소한 한 번은 코드 블록을 실행한다는 특징이 있어요. 즉, 조건이 처음부터 거짓일 경우에도 코드 블록이 한 번 실행됩니다.

### 기본 구조
```kotlin
do {
    // 반복할 코드 블록
} while (참조건)
```

### 예제 코드 설명
```kotlin
fun main() {
    var i = 0  // 변수 i 초기화
    do {
        print(".")  // 출력: 점(.)
        i += 10  // i에 10을 더해 다음 반복으로 이동
    } while (condition(i))  // 조건이 참이면 계속 반복
}
/* 출력 결과:
..........
*/

**핵심 개념 요약:**
- **최소한 한 번 실행**: 루프가 조건이 거짓일 때도 처음에는 무조건 실행됩니다.
- `do-while`은 `while`보다 덜 흔하게 쓰이지만, 특정 상황에서 유용할 수 있어요.

## 간단한 증가 및 감소 연산자

코틀린에서는 `++`와 `--` 연산자를 사용해 간단하게 값을 증가 또는 감소시킬 수 있어요.

### 예제 코드 설명
```kotlin
fun main() {
    var i = 0  // 변수 i 초기화
    while (i < 4) {  // i가 4보다 작을 때까지 반복
        print(".")  // 출력: 점(.)
        i++  // 증가 연산자로 간결하게 처리
    }
}
/* 출력 결과:
....
*/

**핵심 개념 요약:**
- `++`와 `--` 연산자: `i += 1` 대신 사용 가능해 코드를 더 간결하게 만듭니다.

## 반복 범위: `for` 루프

`while` 루프 대신 반복 범위를 다루는 데는 `for` 루프가 더 편리해요. `for` 루프는 특정 범위의 값을 순차적으로 처리해요.

### 기본 구조
```kotlin
for (변수 in 범위) {
    // 반복할 코드 블록
}
```

### 예제 코드 설명
```kotlin
fun main() {
    for (i in 1..3) {  // 1부터 3까지 반복
        println("Hey $i!")  // 각 숫자에 대해 메시지 출력
    }
}
/* 출력 결과:
Hey 1!
Hey 2!
Hey 3!
*/

**핵심 개념 요약:**
- **범위 사용**: `1..3`과 같이 특정 범위를 지정해 쉽게 반복할 수 있어요.
- `in` 키워드: 반복 변수가 각 범위의 값을 차례대로 가지게 됩니다.

### 범위 정의 방법
```kotlin
fun main() {
    val range1 = 1..10  // 1부터 10까지 포함
    val range2 = 0 until 10  // 0부터 9까지 포함 (10은 미포함)
    println(range1)  // 출력: 1..10
    println(range2)  // 출력: 0..9
}
```

- **`..`**: 범위의 양쪽 끝값을 포함합니다.
- **`until`**: 오른쪽 끝값은 포함하지 않습니다.

### 범위를 이용한 합계 계산
```kotlin
fun main() {
    var sum = 0
    for (n in 10..100) {  // 10부터 100까지 반복
        sum += n  // 합계 계산
    }
    println("합계 = $sum")  // 출력: 합계 = 5005
}
```

**핵심 개념 요약:**
- **범위 활용**: 간단하게 범위 내의 모든 값을 순회할 수 있어요.
- `sum += n`: 합계를 쉽게 계산할 수 있게 해줍니다.
