# 16. Objects Everywhere

안녕하세요! 코틀린 프로그래밍을 처음 접하는 분들을 위해 이 챕터를 쉽게 설명해 드릴게요. 이 단원에서는 코틀린에서 **객체(objects)**에 대해 배워볼 거예요. 객체는 데이터를 저장하고 그 데이터를 이용해 다양한 작업을 수행하는 기본적인 구조랍니다. 이해하기 쉽게 하나씩 살펴보도록 하죠!

## 기본 개념 이해하기

### 클래스(Class)와 객체(Object)
- **클래스(Class)**: 새로운 데이터 타입을 정의하는 거예요. 클래스는 속성(값을 저장하는 `val`과 변수 `var`)과 함수(함수들)를 포함해요. 클래스는 사용자가 직접 정의한 타입이라고도 부르죠.
- **멤버(Member)**: 클래스의 속성이나 함수를 의미해요. 클래스의 특성이나 행동을 나타내는 것들이죠.
- **멤버 함수(Member Function)**: 특정 클래스의 객체만 사용할 수 있는 함수예요. 예를 들어, `IntRange` 클래스는 숫자 범위를 다루는 객체죠.

### 객체 생성하기
- **객체 생성**: 클래스의 인스턴스를 만드는 걸 말해요. 쉽게 말해, 클래스를 실제로 사용할 수 있게 만드는 거죠. 예를 들어, `IntRange` 클래스의 객체를 생성하면 각 객체는 독립적인 메모리 공간을 가지게 돼요.

#### 예시: `IntRange` 클래스 사용하기
```kotlin
fun main() {
    // IntRange 클래스의 객체 생성
    val r1 = IntRange(0, 10)  // 0부터 10까지의 범위
    val r2 = IntRange(5, 7)   // 5부터 7까지의 범위

    // 객체 출력
    println(r1)  // 출력: 0..10
    println(r2)  // 출력: 5..7
}
```

- 여기서 `r1`과 `r2`는 각각 독립적인 객체이며, 각자 고유한 메모리 공간을 가지고 있어요. `IntRange` 클래스의 다양한 멤버 함수를 사용할 수 있어요.

## 객체와 멤버 함수 호출하기

### 멤버 함수 호출하기
- **멤버 함수 호출**: 객체에 대해 특정 작업을 수행하는 함수를 호출하는 거예요. 예를 들어, `sum()` 함수를 호출하면 해당 범위의 숫자들을 합산해요.

#### 예시: `IntRange`의 `sum()` 함수 호출
```kotlin
fun main() {
    val r = IntRange(0, 10)  // 0부터 10까지의 범위
    println(r.sum())  // 출력: 55 (0+1+2+...+10의 합)
}
```

- `r.sum()`을 호출하면 `IntRange` 객체 `r`에 정의된 `sum()` 함수가 실행되어 범위 내의 숫자들을 합산해요.

### 코드 완성 기능 활용하기
- **코드 완성(Code Completion)**: IDE(통합 개발 환경)에서 제공하는 기능으로, 객체 뒤에 `.`을 입력하면 해당 객체의 멤버 함수들을 제안해줘요. 예를 들어, `String` 객체의 경우 `reversed()`, `toLowerCase()` 등의 함수를 쉽게 찾을 수 있어요.

#### 예시: `String` 객체의 멤버 함수 사용
```kotlin
fun main() {
    val s = "AbcD"
    println(s.reversed())  // 출력: DcbA
    println(s.toLowerCase())  // 출력: abcD
}
```

- `String` 객체의 다양한 멤버 함수를 쉽게 탐색하고 사용할 수 있어요.

## 숫자 타입 변환

### 타입 변환 예제
- 코틀린에서는 숫자 타입 간의 변환을 명시적으로 수행해요. 예를 들어, `Int`를 `Long`이나 `Double`로 변환할 때는 `.toLong()`, `.toDouble()` 등을 사용해요.

#### 예시: 숫자 타입 변환
```kotlin
fun fraction(numerator: Long, denom: Long) {
    val result = numerator.toDouble() / denom  // Long 타입을 Double로 변환
    println(result)
}

fun main() {
    val num = 1
    val den = 2
    fraction(num.toLong(), den.toLong())  // 출력: 0.5
}
```

- 이 예제에서는 `Int`를 `Long`으로 변환한 후, `Double` 타입으로 다시 변환하여 계산해요.

- **멤버 함수**를 통해 객체의 데이터를 조작하거나 계산해요.
- **코드 완성** 기능을 활용하면 멤버 함수를 쉽게 찾아볼 수 있어요.
