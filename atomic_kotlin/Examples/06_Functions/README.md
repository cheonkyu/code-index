# 06. Functions

안녕하세요! 코틀린 프로그래밍을 처음 접하는 분들을 위해 함수에 대해 좀 더 쉽게 설명해드릴게요. 함수는 프로그램에서 특정 작업을 수행하는 작은 단위라고 생각하면 돼요. 마치 레시피 책에서 특정 요리를 만드는 단계들을 모아놓은 것과 비슷하답니다. 이 단원에서는 함수를 어떻게 정의하고 사용하는지 자세히 알아볼게요.

## 함수의 기본 이해

### 함수란 무엇인가요?
함수는 이름이 붙은 작은 프로그램이에요. 다른 함수에서 이름을 호출해서 실행할 수 있어요. 함수는 여러 작업을 묶어서 프로그램을 정리하고 코드를 재사용하기 쉽게 만들어주는 가장 기본적인 방법이에요. 함수는 입력 정보를 받아서 계산을 하고 결과를 내보내는 역할을 해요.

### 함수의 기본 구조
함수의 기본 형태는 다음과 같아요:

```kotlin
fun 함수이름(매개변수1: 타입1, 매개변수2: 타입2, ...): 반환타입 {
    // 함수 본문의 코드
    return 결과값;
}
```

- **함수이름**: 함수를 부르는 이름이에요. 예를 들어 `multiplyByTwo` 같은 이름이죠.
- **매개변수**: 함수에 전달할 정보의 이름이에요. 예를 들어 `x`는 함수에 들어오는 숫자를 의미해요. 타입은 `: Int`와 같이 명시해요.
- **반환타입**: 함수가 내보내는 결과의 타입이에요. 예를 들어 `: Int`는 정수형을 의미해요.
- **본문 코드**: 중괄호 `{}` 안에 함수가 실제로 수행하는 작업들이 들어가요. 마지막에 `return` 키워드로 결과를 내보내요.

### 예시: `multiplyByTwo` 함수

아래는 간단한 `multiplyByTwo` 함수의 예시예요:

```kotlin
// Functions/MultiplyByTwo.kt
fun multiplyByTwo(x: Int): Int {  // [1] 함수 선언
    println("Inside multiplyByTwo")  // [2] 함수 내부 작업 출력
    return x * 2  // [3] 계산 결과 반환
}

fun main() {
    val r = multiplyByTwo(5)  // 함수 호출 [3]
    println(r)  // 결과 출력
}
/* 출력 결과:
Inside multiplyByTwo
10
*/
```

- **[1]** `fun` 키워드로 함수를 선언하고, 이름 `multiplyByTwo`와 매개변수 `x: Int`, 반환 타입 `Int`를 지정해요.
- **[2]** 함수 내부에서 `println`으로 메시지를 출력해요.
- **[3]** 함수 내에서 `x * 2`를 계산하고 그 결과를 반환해요.

### 매개변수와 인수
- **매개변수**: 함수 안에서 사용할 변수 이름이에요 (예: `x`).
- **인수**: 실제로 함수 호출 시 전달하는 값이에요 (예: `5`). 함수 호출 부분에서 `multiplyByTwo(5)`처럼 사용해요.

### 함수 시그니처
함수 시그니처는 함수의 이름, 매개변수, 반환 타입을 합친 것이라고 생각하면 돼요. 예를 들어 `fun multiplyByTwo(x: Int): Int`가 바로 시그니처예요.

## 반환 타입
함수가 아무런 결과를 내보내지 않을 때는 `Unit` 타입을 반환해요. `Unit`은 단순히 작업 완료를 나타내는 특별한 타입이에요.

### `Unit` 예시
```kotlin
// Functions/SayHello.kt
fun sayHello() {  // 반환 타입이 명시되지 않았지만 Unit으로 반환
    println("Hallo!")
}

fun sayGoodbye(): Unit {  // 반환 타입을 명시
    println("Auf Wiedersehen!")
}

fun main() {
    sayHello()  // Unit 반환
    sayGoodbye()  // Unit 반환
    println()  // main 함수도 Unit 반환
}
/* 출력 결과:
Hallo!
Auf Wiedersehen!
*/
```

### 간결한 표현식 본문
함수 본문을 간결하게 작성할 수도 있어요:

```kotlin
// Functions/MultiplyByThree.kt
fun multiplyByThree(x: Int): Int = x * 3  // 표현식 본문
fun main() {
    println(multiplyByThree(5))  // 출력: 15
}
/* 출력 결과:
15
*/
```

## 함수 호출과 반환 타입 추론
함수 호출 시 간결한 표현식 본문을 사용하면 타입을 명시하지 않아도 Kotlin이 자동으로 추론해요:

```kotlin
// Functions/MultiplyByFour.kt
fun multiplyByFour(x: Int) = x * 4  // 표현식 본문
fun main() {
    val result: Int = multiplyByFour(5)  // 반환 타입이 Int로 추론됨
    println(result)  // 출력: 20
}
/* 출력 결과:
20
*/
```

### 요약
- **함수의 목적**: 코드 재사용과 프로그램 구조화를 위한 작은 프로그램 단위.
- **구성 요소**: 이름, 매개변수, 반환 타입, 본문 코드.
- **인수와 매개변수**: 매개변수는 입력을 위한 플레이스홀더, 인수는 실제 값.
- **간결한 표현식**: 함수 본문을 간결하게 작성할 수 있어요.
- **반환 타입**: `Unit`은 아무런 값을 반환하지 않는 특별한 타입이에요.

함수를 잘 활용하면 코드가 훨씬 깔끔하고 이해하기 쉬워져요. 함수 이름을 명확하게 짓는 것도 중요하니, 작업의 내용을 잘 반영하도록 이름을 정해보세요! 궁금한 점이 있으면 언제든지 물어보세요!
