## 핵심 개념 요약

### 1. **데이터 타입과 변수**
- **코틀린은 정적 타입 언어**로, 변수 타입을 명시적으로 지정해야 합니다. 예를 들어, 정수형 변수를 선언할 때는 다음과 같이 합니다:
  ```kotlin
  var age: Int = 25  // 정수형 변수 age를 선언하고 25로 초기화
  ```
  - 여기서 `: Int`는 변수 타입을 의미합니다. `Int`는 정수형 데이터 타입을 의미해요.

### 2. **함수와 기본 구조**
- **함수 정의**는 코틀린에서 매우 중요한 부분입니다. 함수는 특정 작업을 수행하고 결과를 반환할 수 있어요. 기본 구조는 다음과 같습니다:
  ```kotlin
  fun greet(name: String): String {
      return "안녕하세요, $name님!"
  }
  
  // 함수 호출 예시
  val message = greet("영희")
  println(message)  // 출력: 안녕하세요, 영희님!
  ```
  - `fun` 키워드로 함수를 정의하고, 파라미터 타입과 반환 타입을 지정할 수 있어요. 이 예제에서는 문자열을 입력 받아서 인사말을 반환하는 함수를 보여드렸어요.

### 3. **클래스와 객체**
- **클래스**는 데이터와 함수를 함께 묶어 객체를 생성하는 데 사용됩니다. 기본적인 클래스 정의와 객체 생성 예시는 다음과 같습니다:
  ```kotlin
  class Person(val name: String, val age: Int) {
      fun introduce() {
          println("이름은 $name이고, 나이는 $age살입니다.")
      }
  }

  // 객체 생성 및 사용 예시
  val kim = Person("김철수", 30)
  kim.introduce()  // 출력: 이름은 김철수이고, 나이는 30살입니다.
  ```
  - 클래스 `Person`은 이름과 나이라는 속성을 가지고 있으며, `introduce`라는 메서드를 통해 자기 소개를 할 수 있어요.

### 4. **조건문과 반복문**
- **조건문**과 **반복문**은 프로그래밍에서 매우 자주 사용되는 구조입니다.
  - **조건문 예시**:
    ```kotlin
    if (age > 18) {
        println("성인입니다!")
    } else {
        println("미성년자입니다.")
    }
    ```
  - **반복문 예시**:
    ```kotlin
    for (i in 1..5) {
        println("반복 순서: $i")
    }
    ```
  - 이 구조들은 특정 조건에 따라 코드를 실행하거나, 특정 횟수만큼 코드를 반복하는 데 사용됩니다.
