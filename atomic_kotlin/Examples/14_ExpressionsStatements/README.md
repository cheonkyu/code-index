# 14. ExpressionsStatements

안녕하세요! 이번 챕터에서는 **표현식(Expressions)**과 **문(Statements)**에 대해 알아볼 거예요. 초보자 분들이라면 좀 더 쉽게 이해할 수 있도록 친근하게 설명해드릴게요. 함께 코틀린 프로그래밍의 기본을 탄탄히 다져볼까요?

## 표현식(Expressions)이란?

표현식은 **값을 계산하는 코드 조각**이라고 생각하면 돼요. 쉽게 말해서, 특정 작업을 수행하고 그 결과값을 내는 부분이죠. 예를 들어, 숫자 더하기나 곱하기 같은 연산이 표현식에 해당해요.

### 예시 코드

```kotlin
// 간단한 덧셈 표현식 예시
val result = 5 + 3  // 여기서 `5 + 3`은 표현식이며 결과값은 `8`입니다.

// 변수와 함께 사용된 표현식 예시
val sum = 10 * (2 + 1)  // `10 * (2 + 1)` 역시 표현식으로, 결과는 `30`입니다.
```

### 핵심 포인트
- **결과값을 생성**: 표현식은 항상 결과값을 반환합니다.
- **중괄호 없이도 사용 가능**: 간단한 연산은 중괄호 없이 바로 결과를 변수에 할당할 수 있어요.

## 문(Statements)이란?

문은 **특정 작업을 수행하는 코드 단위**라고 생각하면 됩니다. 표현식도 포함할 수 있지만, 좀 더 넓은 의미에서 여러 작업을 수행할 수 있는 코드 블록이에요. 예를 들어, 할당문이나 제어 구조(if 문, for 문 등)가 여기에 해당해요.

### 예시 코드

```kotlin
// 변수 할당 문 (표현식 포함)
val number = 42  // 여기서 `42`는 표현식이며, 그 결과값이 `number` 변수에 할당됩니다.

// 조건문 예시 (표현식 포함)
if (number > 30) {
    println("큰 숫자입니다!")  // 조건식이 표현식이며, 참일 경우 실행됩니다.
}

// 반복문 예시
for (i in 1..5) {  // `i`는 표현식이며 각 반복마다 새로운 값을 가집니다.
    println("현재 숫자: $i")
}
```

### 핵심 포인트
- **작업 수행**: 문은 코드에서 수행해야 할 다양한 작업을 포함합니다.
- **표현식 포함 가능**: 많은 문들이 표현식을 포함하거나 표현식을 기반으로 작동합니다.

## 요약
- **표현식**: 값을 생성하는 코드 조각 (예: 덧셈, 곱셈 등)
- **문**: 여러 작업을 수행하는 코드 단위 (변수 할당, 조건문, 반복문 포함)
