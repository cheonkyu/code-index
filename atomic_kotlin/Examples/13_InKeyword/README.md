# 13. InKeyword

안녕하세요! 코틀린 프로그래밍 초보자 여러분, 오늘은 코틀린에서 매우 중요한 키워드 중 하나인 `in` 키워드에 대해 배워볼게요. `in` 키워드는 다양한 상황에서 사용되지만, 주로 범위 내에서의 검색이나 반복 작업에서 핵심적인 역할을 합니다. 쉽게 이해하실 수 있도록 단계별로 설명해 드릴게요.

## 기본 개념

`in` 키워드는 주로 두 가지 상황에서 사용됩니다:
1. **범위 내 검색**: 어떤 컬렉션이나 범위 내에서 요소를 찾을 때 사용합니다.
2. **람다 표현식 내에서의 범위 접근**: 람다 표현식이나 컬렉션에서 요소에 접근할 때 사용됩니다.

### 범위 내 검색 예제

`in` 키워드를 사용하여 컬렉션 내에서 특정 요소를 찾는 방법을 볼까요?

```kotlin
fun main() {
    // 리스트 생성
    val numbers = listOf(1, 2, 3, 4, 5)

    // 숫자 3이 리스트에 있는지 확인
    if (3 in numbers) {
        println("3은 리스트에 있습니다!")
    } else {
        println("3은 리스트에 없습니다.")
    }

    // 숫자 6이 리스트에 있는지 확인 (결과는 없음)
    if (6 in numbers) {
        println("6도 리스트에 있습니다!")
    } else {
        println("6은 리스트에 없습니다.")
    }
}
```

**설명:**
- `3 in numbers`는 `numbers` 리스트에 `3`이 존재하는지 확인합니다.
- `if` 문에서 `true`가 반환되면 "3은 리스트에 있습니다!"라는 메시지가 출력됩니다.

### 람다 표현식 내에서의 사용 예제

`in` 키워드는 람다 표현식 내에서도 유용하게 사용됩니다. 특히 컬렉션의 각 요소에 대해 작업을 수행할 때 자주 등장합니다.

```kotlin
fun main() {
    // 숫자 리스트 생성
    val numbers = listOf(10, 20, 30, 40, 50)

    // 각 숫자가 짝수인지 확인하고 결과 출력
    numbers.forEach { number ->
        if (number in 10..50) {  // 10부터 50 사이인지 확인
            println("${number}은 10부터 50 사이의 숫자입니다.")
        } else {
            println("${number}은 범위 밖입니다.")
        }
    }
}
```

**설명:**
- `number in 10..50`은 현재 숫자가 지정된 범위 내에 있는지 확인합니다.
- 각 숫자에 대해 이 조건을 검사하고 결과를 출력합니다.

## 핵심 요약

- **범위 내 검색**: `in` 키워드는 컬렉션이나 범위 내에서 특정 요소의 존재 여부를 쉽게 확인할 수 있게 해줍니다.
- **유연성**: 람다 표현식이나 반복문 내에서 사용하면 코드가 간결해지고 읽기 쉬워집니다.
